Features:

  • [FEATURE]: AND/OR conjunctive expansions. For example, sql = '[[OR UPPER(SOME_COL) = UPPER(:myBind)]]' and binds = { myBind: [1,2] } would result in UPPER(SOME_COL) = UPPER(:myBind) OR UPPER(SOME_COL) = UPPER(:myBind1) with binds = { myBind: 1, myBind1: 2 }.
